Welcome to Cunqa’s documentation!

Cunqa is a Python/C++ library for quantum computing researchers looking to emulate distributed quantum computing using CESGA’s resources. All code available on our GitHub repository.

Check out the README for instructions on how to get started. On Subprogramas you’ll find all cunqa classes.
